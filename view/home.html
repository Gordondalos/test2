Описание задания

Необходимо разработать web-интерфейс состоящий из трех вкладок (слева) и интерфейса авторизации. Допустимо использование фреймворков. Но необходимо в сопроводительном письме указать какой, для какой цели он используется, и почему он был выбран. Вкладки должны обладать следующим функционалом:

1. Первая вкладка содержит форму с кнопкой, а также:

  • ● 3 checkbox, которые делают видимым по одному текстовому полю.
  • ● select c выбором одного из 5 значений: деактивированы все checkbox, активирован 1й, активирован 2й, активирован 1 и 3, активированы все.
  • ● результат выполнения формы возвращает в div находящейся под самой формой текущее время и дату, а также строку c JSON в зависимости от введенных значений следующего вида: { “1” : ”string” , ”3” : ”” }

2. вторая вкладка содержит четырнадцать ячеек для ввода значений:

A: [87] [48] [42] [120] [92] [83] [72]

B: [1] [2] [3] [4] [5] [6] [7]

значение А не может быть меньше B, в случае ввода неверного значения в этом столбце должны быть выделены. Результатом выполнения должны быть 2 графика под содержимым формы. Пример ниже: а также круговой график с суммой выбранных значений.

3. Drag-and-drop Состоит из правой и левой части. В левой части 5 блоков вертикально друг под другом.

  • При двойном клике на блок возможно изменить цвет изменяя значение HTML.
  • Должна быть возможность удалить блок.
  • При изменении положения относительно друг друга в правой части изменяется сожержимое INI:
[block1]
color=#F1F1F1
[block2]
color=#F1F1F1
[block3]
color=#F1F1F1
[block4]
color=#F1F1F1
[block5]
color=#F1F1F1

  • В правой части должны быть 2 кнопки, RESET возвращающее содержимое вкладки в исходное положение,
  • SAVE изменяющее положение блоков на экране согласно содержимому INI файла.

При открытии странички должна быть авторизация admin пароль qwerty123, в самом интерфейсе кнопка выход.
